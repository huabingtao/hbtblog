# koroFileHeader

在 vscode 中用于生成文件头部注释和函数注释的插件

  1. 文件头部添加注释: 快捷键：window：ctrl+alt+i,mac：ctrl+cmd+i
  2. 在光标处添加函数注释: 快捷键：window：ctrl+alt+t,mac：ctrl+cmd+t
